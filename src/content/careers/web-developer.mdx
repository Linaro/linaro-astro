We are looking for an experienced TypeScript web developer to join our team. The successful candidate will be part of a 3-person web development team, responsible for developing and maintaining full-stack applications using TypeScript.

## Responsibilities

- Develop and maintain full-stack applications using TypeScript, HTML, CSS and JavaScript.
- Collaborate with other developers to design, develop and deploy new features.
- Write clean, maintainable, efficient and testable code.
- Troubleshoot and debug applications.
- Participate in code reviews.
- Stay up-to-date with emerging trends and technologies.

## Required Experience

- Proven experience as a developer or similar role.
- Strong, demonstrable proficiency in TypeScript, JavaScript, HTML and CSS.
- Strong, demonstrable proficiency with React and Node.js.
- Experience with writing code tests.
- Familiarity with Git and REST APIs.
- Excellent problem-solving skills.
- Ability to work independently and in a team environment.
- Strong communication skills.

## Advantageous Experience

Demonstrable experience of any of the following would be advantageous:

- Tailwind
- AWS infrastructure
- API development
- Astro
- Next.js
- Infrastructure-as-code tools
- Relational databases and SQL

Hybrid working, with 2 days a week in the Harston (Cambridge) office. You will be expected to occasionally travel to Company events, probably overseas.
